Monitor de Rendimiento Estratégico: Proyecciones, Ventas y Stock

Resumen del Proyecto

Este dashboard permite visualizar el grado de cumplimiento entre la demanda proyectada, las ventas y el abastecimiento. Está diseñado para apoyar la toma de decisiones estratégicas en áreas de planificación comercial, compras y logística, facilitando la identificación de brechas y oportunidades de mejora.

La herramienta se alimenta de datos históricos y proyectados, y presenta indicadores clave que permiten evaluar el rendimiento por categoría, producto y periodo. Su implementación ha contribuido a mejorar la eficiencia operativa y la alineación entre áreas comerciales y operativas.

¿Cuál era el objetivo?

Visualizar el cumplimiento de la demanda en relación con las ventas y el abastecimiento, permitiendo identificar brechas y oportunidades de mejora en la planificación comercial.

Indicadores que seguimos

  • Demanda: Proyección de ventas esperadas.
  • Ventas: Ventas reales registradas.
  • Abastecimiento: Inventario disponible para cubrir la demanda.
  • Órdenes Totales: Número de pedidos procesados.

Tecnologías que usamos

Google Looker Studio para visualización, fuentes de datos conectadas desde hojas de cálculo y Firebase Database.

Resultados obtenidos

Este dashboard ha transformado la gestión del área de planificación al proporcionar información precisa y en tiempo real sobre compras, producción y distribución. Gracias a su implementación, el equipo ha logrado:

  • Reducir el riesgo de desabastecimiento, anticipando necesidades críticas.
  • Optimizar la eficiencia operativa, mejorando la coordinación entre áreas.
  • Tomar decisiones basadas en datos confiables, impulsando una planificación más estratégica.

El resultado: procesos más ágiles, menor desperdicio y una cadena de suministro más robusta.

InsightBot: Chatbot Analítico con LLMs Open-Source

Resumen del Proyecto

Chatbot analítico empresarial que democratiza el acceso a insights de datos mediante inteligencia artificial conversacional, eliminando la necesidad de conocimientos técnicos para el análisis de datos.

Propuesta de Valor

Permite a usuarios no técnicos obtener insights empresariales complejos a través de consultas en lenguaje natural, reduciendo el tiempo de análisis de horas a segundos.

Innovación Técnica

  • LLMs Open-Source: Implementación local de modelos Hugging Face (flan-t5-small) sin dependencias de APIs externas
  • RAG Architecture: Sistema de recuperación aumentada que combina datos estructurados con capacidades conversacionales
  • Interfaz Bilingüe: Soporte nativo español/inglés con detección automática de idioma

Capacidades Demostradas

  • Análisis de 100+ productos across 5 categorías empresariales
  • Procesamiento de 24 meses de datos históricos (2022-2023)
  • Generación de insights sobre ventas, márgenes y tendencias en tiempo real
  • Visualización intuitiva de métricas KPI críticas

Stack Tecnológico

Frontend: Streamlit | Backend: Python 3.9+

AI/ML: Hugging Face Transformers, LangChain

Data: Pandas, NumPy | Deployment: Docker-ready

Casos de Uso

  • Retail Analytics: Análisis de inventario, ventas y tendencias
  • Prototipado IA: Testing de LLMs en escenarios reales
  • Educación: Herramienta didáctica para análisis de datos
  • Enterprise: Solución on-premise para datos sensibles

Resultados obtenidos

Una solución integral que transforma datos complejos en conversaciones inteligentes, posicionándose como bridge entre business intelligence tradicional y AI conversacional moderna.

An Intelligent RPA Workflow

Resumen del Proyecto

Este proyecto es un flujo de trabajo de automatización de procesos robóticos (RPA) con "Human-in-the-Loop", desarrollado en n8n. Su objetivo es optimizar los procesos de validación de datos que, por lo general, son lentos y propensos a errores humanos.

Flujo de trabajo

  1. Extracción de datos: El robot se conecta a una API para obtener datos de tareas recientes.
  2. Validación humana: Envía un correo electrónico interactivo a un gestor de proyectos, quien puede aprobar o rechazar los datos con un solo clic.
  3. Actualización automática: Basado en la decisión del gestor, el robot actualiza automáticamente una hoja de cálculo de Google Sheets para mantener un registro preciso y validado.

Este enfoque combina la eficiencia de la automatización 24/7 con la toma de decisiones críticas de una persona, garantizando la integridad de los datos y liberando horas de trabajo manual.

El problema a resolver

En el mundo de los negocios, la información se mueve rápido. Pero, ¿qué sucede cuando la automatización necesita un toque humano? Los procesos de validación manual se convierten en cuellos de botella que pierden tiempo y dinero. Esta solución elimina ese problema.

El robot desarrollado

Se ha desarrollado un flujo de trabajo en n8n que no solo automatiza, sino que también colabora. Este proyecto es un ejemplo de automatización con "Human-in-the-Loop", una de las tendencias más valiosas de la industria.

Cada día, este robot se encarga de:

  1. Extraer datos frescos: Se conecta a una API externa (jsonplaceholder.typicode.com/todos) para obtener la lista de tareas más reciente.
  2. ¡Alerta inteligente!: Envía un correo electrónico a un gestor de proyectos. El mensaje incluye un resumen de las tareas y un botón para aprobar o rechazar con un solo clic.
  3. Actualizar la base de datos: Basándose en la decisión del humano, el robot actualiza automáticamente una hoja de cálculo de Google Sheets para garantizar que el registro esté siempre perfecto.

¿Por qué es genial?

  • 100% Autónomo: El robot funciona 24/7 sin necesidad de supervisión.
  • Decisiones en segundos: El gestor de proyectos puede aprobar o rechazar datos directamente desde su bandeja de entrada, sin abrir ninguna aplicación.
  • Ahorro de tiempo: Elimina horas de entrada manual de datos.
  • Integridad de datos: Los datos se registran solo después de la validación humana.

Tecnologías que hacen magia

  • n8n: La herramienta de orquestación de código abierto que conecta todo.
  • JSONPlaceholder API: Mi fuente de datos de prueba para simular tareas.
  • Google Sheets: Donde almaceno y gestiono los datos finales.
  • Gmail: El canal de comunicación que une al robot con el humano.
